# Changelog

All notable changes to AutoGBD will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [Unreleased]

### Added
- Initial release of AutoGBD framework
- Intelligent health data harmonization for GBD standards
- Multiple mapping strategies (direct, fuzzy, AI-assisted)
- Comprehensive quality assurance and validation
- Automated report generation with methodological transparency
- Command-line interface and web configuration builder
- Plugin architecture for extensibility
- Complete provenance tracking and audit trails

### Features
- CSV, Excel, and Parquet file support
- ICD-10 to GBD cause mapping with version control
- AI-powered mapping suggestions using transformer models
- Configurable data cleaning rules (8 built-in rules)
- Quality checks with automated scoring (9 validation rules)
- Publication-ready markdown reports
- Streamlit web interface for non-technical users

### Technical
- Python 3.10+ support
- Modern async architecture with Pydantic V2
- Comprehensive test suite (33+ tests)
- Type hints and mypy compatibility
- Docker container support
- CI/CD with GitHub Actions
